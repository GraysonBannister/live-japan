## Lessons Learned

- The LaunchAgent exit code 127 issue recurs daily. Likely due to system restarts or launchd caching. Consider scripting an auto-reload or investigating root cause (e.g., plist validation, path changes).
- When chaining shell commands in exec, use `;` instead of `&amp;&amp;` to avoid parse errors in zsh.
