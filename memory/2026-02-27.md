- **Solution**: Unloaded and reloaded the LaunchAgent:
  ```bash
  launchctl unload ~/Library/LaunchAgents/com.live-japan.ingest.plist
  launchctl load ~/Library/LaunchAgents/com.live-japan.ingest.plist
  ```
- **Verification**: `launchctl list | grep live-japan` now shows exit code 0
